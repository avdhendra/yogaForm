##Getting Started

- First, Clone the repository:

```bash
git clone https://github.com/avdhendra/yogaForm.git
```

- Setup Server
```bash
cd yoga-api
npm install
npm start

server start at port 5000
```

- Setup Frontend
```bash
cd yoga-front
npm install
npm run dev

server start at port 5173
```
